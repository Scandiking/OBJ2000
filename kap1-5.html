1.5 Checkpoint answers

  • 1.5.1 Java ble oppfunnet av et team hos Sun Microsystems. Teamlederen het James Gosling.
  • 1.5.2 En Java-applet er et lite miniprogram som kjører via nettleser. Fordi programmene kunne kjøres via nettleser ble Oracle viden akseptert og brukt. Applets er dessverre ikke lenger tillatt på grunn av sikkerhetsproblemer.
  • 1.5.3 Android har tradisjonelt brukt Java som programmeringsspråk. Fra 2019 er det Kotlin. “Android development will become increasingly Kotlin-first,” Google writes in today’s announcement. “Many new Jetpack APIs and features will be offered first in Kotlin. If you’re starting a new project, you should write it in Kotlin; code written in Kotlin often mean much less code for you–less code to type, test, and maintain.”
