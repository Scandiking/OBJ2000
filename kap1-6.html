1.6 The Java Language Specification, API, JDK, JRE, and IDE

1.6 Checkpoint

  • 1.6.1 What is the Java language specification?
  • 1.6.2 What does JDK stand for? What does JRE stand for?
  • 1.6.3 What does IDE stand for?
  • 1.6.4 Are tools like NetBeans and Eclipse different languages form Java, or are they dialects or extensions of Java?

1.6 Checkpoint answers

  • 1.6.1 Java language specification er en teknisk definisjon av Javas syntakst og semantikk. For eksempel hvordan man alltid må avslutte et statement med et semikolon, vet du. Java language specification definerer dette i spesifikasjonen sin som du kan finne her.
  • 1.6.2 JDK står for Java Development Kit. JRE står for Java Runtime Environment.
  • 1.6.3 IDE står for Integrated Development Environment
  • 1.6.4 Om Java er et språk slik som engelsk, kan man tenke på en IDE (som Eclipse og NetBeans) som en bærer for språket, slik aviser og TV er. NetBeans og Eclipse er ikke "different languages from Java". Du kan kjøre andre språk i disse IDE-ene også, det må ikke være Java.
